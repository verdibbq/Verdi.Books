Verdi.books 🦊📚

Librería virtual argentina especializada en libros y artículos coleccionables. El sitio permite explorar el catálogo, conocer las novedades, contactar para hacer pedidos y conocer la historia del emprendimiento.

- - -

Tecnologías utilizadas

- *HTML5* — estructura semántica de todas las páginas
- *CSS3* — estilos propios (paleta verde-madera), diseño responsive con media queries
- *Bootstrap 5.3* — navbar responsive con menú hamburguesa para móviles
- *WebP / JPEG* — imágenes optimizadas para carga rápida
- *MP4* — video de presentación de productos

- - -
